# LLM Compression SparseGPT Wanda Demo

This Python script demonstrates the concept of Large Language Model (LLM) compression using mock implementations of SparseGPT and Wanda. It simulates pruning a small neural network to reduce its effective size, illustrating the core idea behind making LLMs more efficient for cloud deployment.

## Language

`python`

## How to Run

Run this script using Python: `python llm_compression_demo.py`
